# Work on Feature

Full-cycle feature pipeline: brainstorm → design → plan → review → implement → QA → deploy → archive.

## Arguments

$ARGUMENTS = path to feature folder (e.g., `docs/backlog/active/NNN-feature-name/`)

## Instructions

Invoke `Skill: work-on-feature` and follow its full pipeline for the feature at `$ARGUMENTS`.

The skill file is at `.claude/skills/work-on-feature/SKILL.md` — it contains all 8 phases with detailed instructions.
